Chaff M0 · Cross-Arch Determinism Tester

The one human step (G2): Generate a blob on machine A, Verify it on machine B (different CPU/arch, e.g. ARM ↔ x86). PASS both-ways ⇒ GPT-2 logits are cross-arch deterministic enough for the fixed-point codec ⇒ M1 is greenlit.

Honest threat model. Chaff produces cover that reads as ordinary text to a human or a keyword filter. It is not undetectable and does not defeat a trained steganography classifier that has this exact model. Security rests on the adversary lacking the shared secret and lacking suspicion strong enough to run a matched-model detector.

Self-tests (run before trusting a cross-arch result)

The offline codec self-test round-trips random secrets through the exact codec + crypto that passes the Node gates (G1), using a deterministic mock logit source — it needs no model download. The GPT-2 self-test generates and verifies on THIS machine; it must PASS before a cross-arch attempt is meaningful.
Determinism pins (embedded): backend ONNX-Runtime-Web · WASM EP · fp32 · single-thread. Codec: own fixed-point softmax, snap-to-grid bucketed argsort. Crypto: PBKDF2 ≥600k → AES-256-GCM, fails closed via the GCM tag. A FAIL here localizes to a stage (tokenizer / logit / softmax / bucket). Cross-arch FAIL after a same-machine PASS ⇒ GPT-2 logits diverge across the two architectures at this bucket width — widen the bucket or narrow the product to same-runtime pairs (the M0 decision matrix).
